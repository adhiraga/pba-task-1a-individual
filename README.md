# M-Tix Sentiment Analysis

This project analyzes user sentiment from **19,000+ reviews** of the [M-Tix (Cinema 21)](https://play.google.com/store/apps/details?id=lds.cinema21&hl=id) app on Google Play Store. The full NLP pipeline is built from scratch, starting from raw scraping to feature extraction.

Pipeline: `Scraping` вҶ’ `Tokenization` вҶ’ `Preprocessing` вҶ’ `BoW / N-Grams` вҶ’ `TF-IDF` вҶ’ `POS Tagging`

---

## App Info

| Field | Detail |
|-------|--------|
| App | M-Tix - Cinema 21 |
| App ID | `lds.cinema21` |
| Reviews collected | 19,000+ (Indonesian + English) |
| Sentiment labels | Score вүҘ 4 вҶ’ Positive В· Score вүӨ 2 вҶ’ Negative В· Score = 3 вҶ’ excluded |

## Notebooks

| # | Notebook | Description |
|---|----------|-------------|
| 01 | `01_scraping_tokenization.ipynb` | Scrape reviews from Google Play Store, tokenize, lowercase, remove punctuation |
| 02 | `02_preprocessing_eda.ipynb` | Remove stopwords, stem with PySastrawi, label sentiment, visualize with pie & bar chart |
| 03 | `03_bow_ngrams.ipynb` | Bag of Words, bigram, and trigram feature extraction |
| 04 | `04_tfidf.ipynb` | TF-IDF vectorization, top words per sentiment class |
| 05 | `05_pos_tagging.ipynb` | POS tagging with NLTK, POS distribution analysis |

## How to Run

**Option 1 - Google Colab (recommended)**

Open each notebook directly in Colab and run all cells top to bottom.

**Option 2 - Local (VS Code)**

```bash
git clone https://github.com/adhiraga/pba-task-1a-individual.git
cd pba-task-1a-individual
pip install -r requirements.txt
```

Run notebooks in order: `01` вҶ’ `02` вҶ’ `03` вҶ’ `04` вҶ’ `05`. Notebooks `03` to `05` depend on `mtix_preprocessed.csv` generated by notebook `02`.
